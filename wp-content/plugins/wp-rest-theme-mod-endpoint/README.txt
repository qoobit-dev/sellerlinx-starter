== Description ==
Adds Endpoint for current active theme customier fields. 'wp-json/themes/v1/customizer-fields'
== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `customizer-end-point.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Access endpoint with /wp-json/themes/v1/customizer-fields
