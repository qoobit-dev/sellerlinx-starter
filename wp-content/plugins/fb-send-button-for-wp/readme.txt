This plugin adds a facebook/messenger send button to your WordPress posts, pages, before and after content and gives you a shortcode as well.


== Description ==


**What does Facebook Send button do**
The send button allows your users to share your blog posts/pages with their friends. They can also use the send button to send the message to Facebook Groups that they are part of. Think of this as an email-this button on steroids because it has the social touch to it. 

**Features**
- Add Facebook Send Button to Posts and Pages
- Add Facebook Send Button to Before and after Content
- Add Facebook Send Button with shortcode `[wpfbs]`


**Shortcode**

You can use `[wpfbs]` shortcode to add Facebook Send Button anywhere you want.

Example Usage: `[wpfbs link="http://example.com/"]`

== Installation ==

Install/Activate the plugin and go to Settings > Facebook Send Button page to configure it.

e.g.

1. Upload `\fb-send-button-for-wp\` to the \`/wp-content/plugins/\` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Activate the plugin and go to `Settings > Facebook Send Button` page to configure it.

== Frequently Asked Questions ==

= How can I configure this plugin? =

Activate the plugin and go to `Settings > Facebook Send Button` page to configure it.

= How to add the shortcode? =

You can use `[wpfbs]` shortcode to add Facebook Send Button anywhere you want.
- Example Usage: `[wpfbs link="http://example.com/"]`
This can be used inside the content area where ever the shortcodes work.
